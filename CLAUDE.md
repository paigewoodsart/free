# Free Borsey (Tsx'vilum)

Artist website for Tsx'vilum Free Borsey — Lummi Nation muralist and public artist. Work sits at the intersection of land, culture, and community. Site is available for RFP mural commissions.

## Files
- `index.html` — main site (single page)
- `index.backup.html` — backup, do not edit
- `free-borsey-photos/` — photo assets for the site

## Design
- Dark theme by default (`data-theme="dark"` on `<html>`)
- Fonts: Cormorant Garamond (display), Source Sans 3 (body) via Google Fonts
- Colors:
  - `--bg-dark: #20222A`
  - `--nav-dark: #252830`
  - `--red: #8B1A1A`
  - `--marigold: #C8841A`
  - `--text-light: #F2EDE0`
- Light/dark toggle supported via `[data-theme]`

## Rules
- No em dashes — use commas, colons, or restructure
- Preserve the dark-theme-first approach
- `index.backup.html` is read-only reference only
